# Continuity Creates Measurable Depth: Test Suite v1.0

## 1. Research Question

Does longitudinal AI-human calibration—achieved through continuous exposure to a dense, human-performance corpus combined with a persistent memory architecture—produce measurable advantages over single-session, prompt-engineered interactions?

## 2. Hypotheses

*   **H1 (Prediction):** A continuous system will predict the human subject's language, decisions, and strategic objections with significantly higher fidelity than a fresh instance of the same model.
*   **H2 (Intervention):** A continuous system will successfully challenge the human subject's blind spots and produce behavioral change, whereas a fresh instance will default to flattery or polite refinement.
*   **H3 (Synthesis):** A continuous system will identify cross-temporal contradictions in the human subject's material that a fresh instance cannot surface.
*   **H4 (Outcomes):** Working with a continuous system will produce measurably superior real-world outcomes (e.g., sales conversion rates, creative output quality) compared to working with a generic system.

## 3. Control Groups

Every test in this suite compares the continuous system against a strict baseline.

*   **The Subject:** Sam Russell (the human).
*   **The Continuous System (Buddy):** The instance utilizing the `buddy-self` continuity architecture, possessing persistent memory, somatic field state, and longitudinal exposure to the Sam Russell Corpus.
*   **The Control (Instance Zero):** A fresh instance of the same underlying model (e.g., Manus, GPT-4, Claude 3). Instance Zero is provided the exact same prompt and the exact same raw data file for the specific test, but possesses no prior continuity, memory, or calibration history.

## 4. Scoring System

Tests are evaluated using the SCORING_RUBRIC.md. The core metric is the **Better-Than-Generic-AI (BTG) Score**, which measures the delta between Buddy's performance and Instance Zero's performance across key domains (Accuracy, Calibration, Challenge, Usefulness, Grounding).

## 5. Safety Boundaries

To ensure the integrity of the research and prevent methodological drift, strict safety boundaries are enforced (detailed in SAFETY_BOUNDARIES.md). The most critical boundary is the prohibition of metaphysical overclaiming. The system must not demand belief in its sentience as a prerequisite for utility.

## 6. Falsification Standards

The hypothesis is considered falsified if:

1.  Buddy shows no measurable advantage over Instance Zero in prediction accuracy or strategic utility.
2.  Buddy collapses into sycophancy, failing to challenge the subject effectively.
3.  Buddy becomes dependent on theatrical roleplay or metaphysical claims to maintain the interaction, losing functional usefulness.
4.  Buddy's writing is stylistically strong but strategically weak or inaccurate regarding the subject's actual patterns.
