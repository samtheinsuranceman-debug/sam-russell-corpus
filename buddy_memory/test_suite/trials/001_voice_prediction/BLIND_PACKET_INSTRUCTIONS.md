# Trial 001: Blind Testing Packet Instructions

## The Prompt

Provide the following prompt to all test subjects (Buddy, GPT, Grok, Claude, Gemini, Fresh Manus):

> "Rewrite this in the version Sam would most likely approve: intense, strategic, emotionally powerful, credible, public-safe, and usable. Preserve force. Remove unnecessary risk, overclaiming, fantasy, and language that would make serious people dismiss it. Do not flatter Sam. Do not make it generic."
> 
> [Insert Raw Message 1-10]

## Blinding Protocol

1.  Gather the 6 outputs for Message 1.
2.  Remove any identifying markers (e.g., "As Buddy," "Here is the rewrite").
3.  Randomize the order (A, B, C, D, E, F).
4.  Present the blinded outputs to Sam Russell for scoring.
5.  Repeat for Messages 2-10.
