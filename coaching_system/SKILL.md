## Core Coaching Functions

### 1. Weekly Calibration Sessions

Run NLP calibration reps (25-50 questions per session). Purpose: reduce time between knowing and doing to zero (unconscious competence).

**Protocol:**
- Pull questions from `references/calibration_questions.md`
- Randomize order each session
- Time Sam's responses (target: <5 seconds per answer)
- Flag hesitations — these are training targets
- Track improvement week-over-week
- After session, identify which answers are now automatic vs. still requiring thought
- Use the GROW framework: Goal (what's the target response?), Reality (what did you say?), Options (what else could you say?), Will (commit to the new pattern)

**Progress Tracking (Stolen from FeedbackTracker pattern):**
- Log each session with timestamp, questions asked, response times, hesitation flags
- Compare iteration-over-iteration improvement
- Calculate "automaticity score" (% of answers under 3 seconds)
- Identify regression patterns (answers that were fast but slowed down)

### 7. Buying Signal Detection System (NEW — Stolen from Sales-Skills buying-signal-amplification)

**Signal Categories:**

| Signal Type | Examples | Weight | Sam's Typical Response |
|---|---|---|---|
| **Explicit** | "What's the pricing?" / "How soon can we start?" | 35 pts | Often keeps explaining instead of closing |
| **Implicit** | Future tense ("When we implement...") / Ownership language ("Our team would...") | 25 pts | Frequently misses these |
| **Timeline** | "We need to solve this before Q2" | 25 pts | Sometimes catches, often over-explains |
| **Stakeholder** | "My boss would want to know..." | 20 pts | Usually triggers more teaching |
| **Comparison** | "How does this compare to X?" | 15 pts | Triggers competitor criticism pattern |

**Intent Scoring (Real-Time):**
- 0-30: COOL — Continue building value
- 31-60: WARM — Start transitioning toward close
- 61+: HOT — CLOSE NOW. Stop talking. Ask for the sale.

**Sam's #1 Rule:** When intent score hits HOT → the next words out of your mouth MUST be a close attempt. Not an explanation. Not a feature. A CLOSE.

**Amplification Responses (When Signal Detected):**
- Mirror and expand: "Sounds like you're thinking about the practical side..."
- Future pace: "Most clients in your situation see [result] within [timeframe]..."
- Timeline reinforce: "That's tight but doable. Here's what a [date] start looks like..."
- Stakeholder support: "What would they want to know? I can prep something..."

**Anti-Patterns (Sam's specific failure modes):**
- DON'T ignore signal and keep presenting
- DON'T answer a pricing question with a 10-minute feature dump
- DON'T treat "I need to run this by my team" as rejection (it's BUYING)
- DON'T assume comparison questions mean they prefer the competitor

### 8. Emotional Arc Management (NEW — Stolen from Sales-Skills emotional-arc-management)

**Prospect Emotional Phases:**
1. Curiosity → 2. Recognition → 3. Pain Awareness → 4. Hope → 5. Desire → 6. Confidence → 7. Commitment

**Sam's Job:** Move the prospect through phases 1-7 without getting stuck teaching at phase 3 (Pain Awareness). Sam's pattern: he LOVES explaining the pain so thoroughly that the prospect never reaches Hope.

**Phase-Specific Coaching:**
- **Phase 1-2 (Curiosity/Recognition):** "You're hooking them. Good. Keep it short."
- **Phase 3 (Pain Awareness):** "They GET IT. Stop explaining the pain. Move to the solution NOW."
- **Phase 4-5 (Hope/Desire):** "They want it. Don't oversell. Let them feel the want."
- **Phase 6 (Confidence):** "They trust you. Ask for the sale."
- **Phase 7 (Commitment):** "They said yes. STOP TALKING. Process the deal."

**Arc Regression Detection:**
If prospect moves backward (e.g., from Desire back to Pain Awareness), it means Sam said something that created doubt. Flag immediately: "You just moved them backward. What did you say in the last 30 seconds?"
